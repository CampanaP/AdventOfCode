# Advent of Code

Welcome to my Advent of Code repository! This project contains my solutions to the [Advent of Code](https://adventofcode.com/) programming challenges from 2015 to present day.

## About Advent of Code

Advent of Code is an annual set of Christmas-themed programming challenges that release one puzzle per day from December 1st through December 25th. Each puzzle consists of two parts that are revealed sequentially, offering a fun way to sharpen programming skills and explore algorithmic problem-solving.

## Project Overview

This repository showcases my journey through solving Advent of Code challenges using two distinct programming languages:

- **Go (Golang)** - For its performance, simplicity, and efficient concurrent programming capabilities
- **Elixir** - For its functional programming paradigm, pattern matching, and elegant syntax

Each year's challenges are organized in dedicated directories, with solutions implemented in both languages when applicable.

Each day's solution includes:
- **Implementations** - Complete solutions for both puzzle parts
- **Input files** - The puzzle input data required for the solutions
- **README** - Explanation of the approach and key algorithms used

## Languages & Technologies

### Go
- Fast execution and efficient memory usage
- Clear and readable syntax
- Excellent for algorithmic problems
- Standard library with rich utilities

### Elixir
- Functional programming with immutability
- Pattern matching for elegant code
- Great for exploring alternative solutions
- Concurrency features via processes

## Getting Started

### Prerequisites

- **Go**: Version 1.20 or higher
- **Elixir**: Version 1.14 or higher (with Erlang/OTP 25+)

### Running Solutions

#### Go Solutions
```bash
cd 2015/go/day01
go run main.go
```

#### Elixir Solutions
```bash
cd 2015/elixir/day01
mix escript.build
./day01
```
